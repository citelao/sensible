{% comment %} Jekyll auto-populates previous and next for collections, which is a nice thing. However, it is impossible to get file name ordering if your collection items include dates. YAML and Jekyll also work to make it impossible to override next to some `void` type or add an extra property to it (`ignore`). Instead, use overall properties `ignore_previous` and `ignore_next`. {% endcomment %} {% if page.real_previous == nil and page.previous.url and page.ignore_previous != true %} « {{page.previous.title}} {% endif %} {% if page.real_previous %} « {{page.real_previous.title}} {% endif %} {% if page.real_next == nil and page.next.url and page.ignore_next != true %} {{page.next.title}} » {% endif %} {% if page.real_next %} {{page.real_next.title}} » {% endif %}
